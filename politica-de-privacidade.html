Política de Privacidade

A Docs Express Brasil respeita sua privacidade e está comprometida com a proteção dos seus dados pessoais, em conformidade com a LGPD.

Coleta de informações

Coletamos apenas as informações necessárias para a prestação dos nossos serviços, como nome, telefone e dados enviados voluntariamente pelo cliente via WhatsApp. Não solicitamos informações além do estritamente necessário para elaborar o documento ou realizar a análise contratual solicitada.

Uso das informações

As informações são utilizadas exclusivamente para elaboração de documentos, análise contratual, atendimento ao cliente e melhoria dos nossos serviços. Nenhum dado é utilizado para fins de marketing sem consentimento expresso.

Compartilhamento de dados

Não compartilhamos seus dados com terceiros, exceto quando estritamente necessário para cumprimento de obrigações legais ou determinação judicial. Os dados de terceiros mencionados em contratos enviados para análise são tratados com o mesmo nível de sigilo e utilizados exclusivamente para a prestação do serviço contratado.

Segurança

Adotamos medidas técnicas e organizacionais para proteger suas informações contra acesso não autorizado, perda ou uso indevido. O atendimento via WhatsApp utiliza a criptografia de ponta a ponta da plataforma.

Seus direitos (LGPD)

Em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018), você pode a qualquer momento solicitar: acesso aos seus dados, correção de informações incorretas, exclusão de dados pessoais, portabilidade ou revogação de consentimento. Para exercer esses direitos, entre em contato pelo nosso WhatsApp oficial.

Retenção de dados

Os dados são mantidos pelo tempo necessário para a prestação do serviço e cumprimento de obrigações legais. Após esse período, são descartados de forma segura.

Contato
